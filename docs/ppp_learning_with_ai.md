## Learning Topic 1

# Sports Betting Analytics

---

## Sports Betting Analytics - Goal

Develop a comprehensive understanding of the **mathematics, statistics, and data analysis** techniques used in sports betting.

---

## Why This Topic?

- ✅ Combines statistics, probability, and real-world decision-making
- ✅ Practical application of data analysis skills
- ✅ Personal interest in college football and sports analytics
- ✅ Develops quantitative thinking applicable to many domains

---

## Key Learning Objectives

1. Understand how sportsbooks calculate and set odds
2. Master expected value (EV) calculations for betting scenarios
3. Learn statistical models (Elo ratings, regression models)
4. Interpret betting lines, spreads, and over/unders
5. Apply data analysis to identify value bets
6. Understand bankroll management mathematics

---

## Learning Approach with AI

- **AI as Tutor:** Use Claude/ChatGPT to explain statistical concepts with sports betting context
- **Hands-on Practice:** Build Python scripts with AI assistance to calculate EV and analyze data
- **Model Building:** Learn to create simple predictive models with AI guidance
- **Iterative Learning:** Ask progressively deeper follow-up questions to build understanding

---

## Sports Betting - Deliverables

- Python scripts for EV calculations and betting simulations
- Analysis of historical sports data
- Documentation of key concepts learned
- Simple predictive model implementation

---

## Sports Betting - Success Metrics

✅ Ability to explain core concepts (odds, EV, probability) without AI assistance
✅ Successfully implement working analysis scripts
✅ Demonstrate understanding through code that produces accurate results
✅ Apply concepts to analyze real betting scenarios

## Learning Topic 2

# Stock Market Analysis

---

## Stock Market Analysis - Goal

Gain practical knowledge of stock market analysis techniques, including both **fundamental and technical analysis**, to make informed investment decisions.

---

## Why This Topic?

- ✅ Essential life skill for financial planning
- ✅ Connects to capstone project theme (financial wellness)
- ✅ Combines data analysis with financial literacy
- ✅ Practical application with real-world impact

---

## Key Learning Objectives

1. Understand stock market fundamentals (exchanges, order types, market mechanics)
2. Learn fundamental analysis: financial statements, key metrics (P/E, revenue growth)
3. Learn technical analysis: candlestick charts, trends, indicators (RSI, moving averages)
4. Understand portfolio diversification principles
5. Learn basics of options trading and pricing
6. Explore algorithmic/quantitative trading strategies

---

## Learning Approach with AI

- **AI as Tutor:** Break down complex financial concepts into digestible explanations
- **Real-World Examples:** Analyze actual company financials and stock charts with AI guidance
- **Coding Practice:** Write Python scripts using libraries like yfinance for data analysis
- **Paper Trading:** Create and test trading strategies on historical data with AI help
- **Progressive Learning:** Start with fundamentals, build to advanced concepts

---

## Stock Market - Deliverables

- Python scripts for stock data analysis and visualization
- Technical indicator calculations and chart analysis
- Fundamental analysis of selected companies
- Paper trading strategy with performance evaluation
- Documentation of learning journey

---

## Stock Market - Success Metrics

✅ Ability to read and interpret financial statements independently
✅ Successfully implement technical indicators in code
✅ Demonstrate understanding through working analysis tools
✅ Make informed analysis of real stocks using learned techniques

# Learning with AI

## Overview

This repository documents my journey of learning new topics using AI tools effectively. The goal is to demonstrate that I can use AI as a tutor and learning assistant to gain real understanding of unfamiliar subjects. Each topic includes what I want to learn, why I chose it, and how I plan to use AI tools throughout the learning process.

---

## Topic 1: Sports Betting Analytics

### Why I Want to Learn This

I follow college football and have a casual interest in sports betting, but I don't have a deep understanding of how odds are calculated, what makes a betting strategy mathematically sound, or how data analysis plays into it. I want to understand the math and statistics behind sports betting — things like probability, expected value, line movement, and how sportsbooks set odds. This is a topic that combines statistics, data analysis, and real-world decision-making, which are all valuable skills.

### What I Want to Learn

- How sportsbooks calculate and set odds
- The concept of expected value (EV) and how it applies to betting
- Common statistical models used in sports analytics (e.g., Elo ratings, regression models)
- How to read and interpret betting lines, spreads, and over/unders
- The role of data analysis in identifying value bets
- Bankroll management and the math behind it

### How I Will Learn with AI

- **Claude / ChatGPT:** Use as a tutor to explain statistical concepts like probability, expected value, and variance in the context of sports betting. Ask follow-up questions to deepen understanding.
- **AI-assisted coding:** Use AI to help me write Python scripts that calculate expected value, simulate betting outcomes, and analyze historical sports data.
- **Concept exploration:** Have AI explain how professional sports analysts and oddsmakers use models, and walk me through building a simple predictive model step by step.

### What Proof I'll Produce (Deliverables)

- Python scripts for EV calculations and betting simulations
- Analysis of historical sports data with visualizations
- Documentation of key concepts learned (probability, odds, statistical models)
- Simple predictive model implementation
- Weekly reflections on learning progress

## Topic 2: Stock Market Analysis

### Why I Want to Learn This

Investing is an important life skill, and I want to understand how the stock market works beyond the basics. I'm interested in learning how people analyze stocks to make informed decisions — both fundamental analysis (looking at a company's financials) and technical analysis (reading charts and patterns). This connects to my broader interest in using data to make better financial decisions, which is also related to my capstone project's theme of financial wellness.

### What I Want to Learn

- The fundamentals of how the stock market operates (exchanges, market makers, order types)
- Fundamental analysis: how to read financial statements, evaluate P/E ratios, revenue growth, and other key metrics
- Technical analysis: how to read candlestick charts, identify trends, and understand indicators like RSI and moving averages
- How portfolio diversification works and why it matters
- The basics of options trading and how options are priced
- How algorithmic and quantitative trading strategies work at a high level

### How I Will Learn with AI

- **Claude / ChatGPT:** Use as a tutor to break down complex financial concepts into understandable explanations. Start with fundamentals and progressively ask deeper questions.
- **AI-assisted research:** Ask AI to explain real-world examples — for instance, walk through analyzing a specific company's financials or interpreting a real stock chart pattern.
- **AI-assisted coding:** Use AI to help me write Python scripts that pull stock data (using libraries like yfinance), calculate technical indicators, and visualize trends.
- **Simulated practice:** Have AI help me create a paper trading strategy and evaluate its performance over historical data.

### What Proof I'll Produce (Deliverables)

- Python scripts for stock data analysis and visualization
- Technical indicator calculations and chart analysis examples
- Fundamental analysis of 2-3 selected companies
- Paper trading strategy with performance evaluation
- Weekly reflections and learning documentation

## How I'll Use AI Responsibly

### Verification & Understanding

- Never blindly copy AI-generated code without understanding it
- Test all AI-provided code and verify outputs manually
- Ask follow-up questions until I truly understand concepts
- Implement concepts myself to validate understanding

### Prompting Strategy

- Start with broad concept questions, then drill down into specifics
- Ask AI to explain "why" not just "how"
- Request multiple approaches to solve problems
- Have AI explain trade-offs and limitations

### Citation & Attribution

- Document when AI tools help generate code (comments in scripts)
- Cite external resources AI recommends in `resources.md` files
- Keep notes on which concepts required AI help vs. independent work
- Be transparent about the learning process in reflections
